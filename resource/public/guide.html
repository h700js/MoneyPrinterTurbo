1. 🤖 What is MoneyPrinterTurbo?

MoneyPrinterTurbo is an AI-powered tool that automatically creates short videos. You just type a topic — like "Top 10 AI Tools 2026" — and the AI does everything:

  • 🧠 Writes the script using AI (like ChatGPT or DeepSeek)
  • 🎥 Finds video clips from Pexels or Pixabay matching your topic
  • 🗣️ Creates voiceover using Text-to-Speech (TTS) technology
  • 📝 Adds subtitles with customizable fonts and colors
  • 🎵 Adds background music automatically
  • ⚡ Renders a complete HD video ready to upload
💡 Who is this for?
TikTok & Reels creators, digital marketers, educators, news channels, affiliate marketers — anyone who needs high-quality short videos fast.

2. 🔄 How it Works — The Big Picture

The video creation process follows 4 simple stages:

Stage 1: Script Generation
You enter a topic. AI (via LLM provider like DeepSeek or OpenAI) writes a script and suggests search keywords.
Stage 2: Material Collection
The app searches Pexels or Pixabay for video clips related to your keywords. You can also upload your own videos.
Stage 3: Assembly
The app combines clips, adds voiceover (TTS), subtitles, and background music. You control duration, transitions, styles.
Stage 4: Rendering
A high-definition MP4 video is created. Watch it in the browser or download it.
⚠️ Important: This app needs internet connection to call AI APIs. Only Ollama + Local files work offline.

3. ⚡ Quick Start: First Video in 5 Minutes

Follow these 4 simple steps to create your first video:

Step 1: Enter a Topic

In the left panel, type a video subject. Examples:

  • "Top 10 AI Tools 2026"
  • "How to make a chocolate cake"
  • "Crypto news today"
  • "5 tips for better sleep"

Step 2: Generate Script

Click the button "Generate Video Script and Keywords". The AI will write a full script and suggest keywords for finding video clips. Wait a few seconds.

Step 3: Adjust Settings (Optional)

You can skip this step if you want — the default settings work fine. But if you want to customize:

  • Right panel: Choose subtitle font, color, position
  • Middle panel: Pick video ratio (9:16 for TikTok, 16:9 for YouTube)
  • Middle panel: Choose a voice for narration

Step 4: Generate!

Click the big "Generate Video" button at the bottom. Wait 1-3 minutes while the AI finds clips, adds voice, and renders your video. When done, you'll see the video player appear!

💡 Tip: Make sure you have entered at least one LLM API key and one Video Source API key in Basic Settings (click the expandable panel at the top).

4. 🔑 Getting API Keys (Step by Step)

To use MoneyPrinterTurbo, you need at least two API keys: one for AI script writing (LLM), and one for video material (Pexels or Pixabay). Here's how to get them free:

Option A: DeepSeek (Best for Indonesia — No VPN) ✅

  1. Go to platform.deepseek.com
  2. Click "Sign Up" and create an account (email + password)
  3. Go to "API Keys" page
  4. Click "Create API Key" — copy the key (starts with sk-)
  5. Paste it in Basic Settings → LLM Settings → API Key
  6. Set Base URL to https://api.deepseek.com
  7. Set Model Name to deepseek-chat
💰 Cost: DeepSeek gives free credits when you sign up. Very cheap after that (~$0.14 per 1M tokens).

Option B: OpenAI (Needs VPN)

  1. Go to platform.openai.com
  2. Sign up, add payment method
  3. Create an API key
  4. Set Model Name to gpt-4o or gpt-3.5-turbo

Option C: Groq (100% Free)

  1. Go to console.groq.com/keys
  2. Sign up with Google or GitHub
  3. Create an API key
  4. Set Model Name to llama-3.3-70b-versatile
  5. Set Base URL to https://api.groq.com/openai/v1
💰 Cost: Free! But limited requests per minute.

Getting a Pexels API Key (Free Video Source)

  1. Go to pexels.com/api
  2. Click "Join Pexels" or sign up with Google/Facebook
  3. Go to "Your API Key" page
  4. Copy the key (a string of letters and numbers)
  5. Paste it in Basic Settings → Video Source Settings → Pexels API Key
💰 Cost: Free! 200 requests per hour. More than enough for personal use.

Getting a Pixabay API Key (Alternative Free Video Source)

  1. Go to pixabay.com/api/docs
  2. Sign up for a Pixabay account

7. 🎬 Video Settings — Customizing the Visuals

The middle panel controls how your video looks.

Video Source

  • Pexels (recommended): Free HD stock videos. Need API key.
  • Pixabay: Alternative free stock videos. Need API key.
  • Local file: Upload your own videos or images from your computer.
  • TikTok / Bilibili / Xiaohongshu: Not yet available.

Video Concat Mode

  • Random: Video clips are shown in random order. More varied.
  • Sequential: Clips follow the same order they were found.

Video Transition Mode

How one clip changes to the next:

  • None: Direct cut (no effect)
  • Shuffle: Random transition effect
  • FadeIn / FadeOut: Fade to black and back
  • SlideIn / SlideOut: Slide new clip in

Video Ratio

  • Portrait 9:16 — For TikTok, Instagram Reels, YouTube Shorts
  • Landscape 16:9 — For YouTube videos, presentations

Clip Duration

How long each video clip plays. 3-5 seconds is recommended for fast-paced short videos.

Number of Videos

Generate 1 to 5 videos at the same time. Each video will have slightly different clips and music.

Advanced Video Settings

Video Encoder: Choose between:

  • libx264 (CPU) — Works everywhere, slower but safe
  • h264_nvenc (NVIDIA) — Much faster if you have an NVIDIA GPU
  • h264_amf (AMD) — Faster with AMD graphics card
  • h264_qsv (Intel) — Faster with Intel integrated graphics
💡 Tip: If you're not sure, leave it as libx264. It works on all systems.

8. 🔊 Audio Settings — Voice & Music

Still in the middle panel, below Video Settings.

TTS Servers (Text-to-Speech)

This is what reads your script aloud. Options:

  • Azure TTS V1 (Default) — ✅ Free, no setup needed. Good quality.
  • Azure TTS V2 — Better quality, but needs Azure API key + region.
  • SiliconFlow TTS — Alternative TTS, needs API key from SiliconFlow.
  • Google Gemini TTS — Uses your Gemini API key. Good quality.
  • Xiaomi MiMo TTS — From Xiaomi, needs MiMo API key.
  • No Voice — Skip voiceover (video will be silent or use only BGM).

Speech Synthesis Voice

Choose a voice. Available voices depend on your TTS server. For Azure TTS V1, you can choose from many languages and genders.

Speech Volume & Rate

  • Volume: 1.0 is normal. Lower for background narration.
  • Rate: 1.0 is normal speed. 1.2 is slightly faster.

Custom Audio File

Upload your own audio (MP3, WAV, etc.). If you upload, the TTS voice will be skipped and your audio will be used directly.

Background Music

  • No Background Music: Only voiceover, no BGM.
  • Random Background Music: The app picks a random song from built-in tracks.
  • Custom Background Music: Type the filename (place your MP3 in the resource/songs folder).
🎵 Music tip: Keep background music volume at 0.2-0.3 so the voiceover stays clear and easy to hear.

9. 📝 Subtitle Settings — Adding Text

The right panel controls subtitles (the text that appears on screen).

Enable Subtitles

Turn subtitles on or off. If off, the rest of the settings below don't matter.

Subtitle Font

Choose from available fonts. The app comes with several fonts built-in.

Subtitle Position

  • Bottom: Most common — subtitles at the bottom of the screen.
  • Center: In the middle (great for highlighting key words).
  • Top: At the top.
  • Custom: Choose a percentage from the top (e.g. 70% = near bottom).

  • Font Size: 30 (small) to 100 (large). 60 is a good default.
  • Font Color: Pick any color. White (#FFFFFF) is safest.
  • Stroke Color & Width: Adds a border around text for readability.

Subtitle Background

Enable a background behind the text so it's readable even on light-colored video.

  • Background Color: Usually black (#000000) with transparency.
  • Rounded Background: Makes the background pill-shaped instead of rectangular.

10. 🚀 Generating and Downloading Videos

Before You Click Generate

Checklist:

  • ✅ LLM API key is filled (Basic Settings)
  • ✅ Video Source API key is filled (Pexels or Pixabay)
  • ✅ Video subject or script is entered
  • ✅ Internet connection is working

Click "Generate Video"

The big button at the bottom. The process takes 1-5 minutes depending on video length and internet speed.

What happens during generation:

  1. The app calls the LLM to write a script (if not already provided)
  2. The app searches Pexels/Pixabay for video clips
  3. The app generates voiceover audio using TTS
  4. The app combines everything — clips, audio, subtitles, music
  5. The app renders the final MP4 video file

After generation

You'll see a video player appear. You can watch the video directly in the browser. The video is also saved to the storage/tasks/ folder on your server.

11. 💡 Pro Tips & Best Practices

Writing Good Topics

  • Be specific: "How to make a chocolate cake" is better than "Cake"
  • Include numbers: "Top 5 AI Tools" is more clickable than "AI Tools"
  • Add a target: "For beginners" or "In 2026" helps AI focus

Getting Better Voiceover

  • Keep script between 50-100 words for a 30-second video
  • Use simple sentences — AI reads them more naturally
  • Add punctuation (periods, commas) — it helps TTS sound natural
  • Test different voices — some sound more natural than others

Video Quality Tips

  • Use Pexels as video source — it has the best quality clips
  • Set clip duration to 4 seconds — not too short, not too long
  • Use Random transition for a professional look
  • Keep background music at 20-30% volume
  • Enable subtitles — many viewers watch without sound

Batch Generation Strategy

Set "Number of Videos" to 3-5 to create multiple versions. Each will have different clips and music. Post them across different platforms to see which performs best.

Saving Time

  • Save your API keys — the app remembers them
  • Reuse scripts — edit slightly for new videos
  • Use the same settings — once you find what works, stick with it

12. 🔧 Troubleshooting — Common Problems

"Please Enter the LLM API Key"

Go to Basic Settings, make sure you've selected an LLM Provider and entered a valid API key.

"Please Enter the Pexels API Key"

Go to Basic Settings → Video Source Settings, enter your Pexels API key. Get one free at pexels.com/api.

Video generation fails / stuck

  • Check your internet connection
  • Make sure your API keys are correct
  • Try a shorter video topic with fewer keywords
  • Reduce "Number of Videos" to 1
  • Check the log panel — it shows what's happening behind the scenes

Subtitles not appearing

Make sure "Enable Subtitles" is checked in the right panel.

No voice / audio

  • Check TTS server is selected (not "No Voice")
  • Make sure you chose a voice in "Speech Synthesis"
  • Check speech volume is not set to 0

Video clips not found

  • Your keywords might be too specific. Try broader terms.
  • Check that your Pexels/Pixabay API key is valid.
  • Switch to a different video source.

"Error from LLM" message

  • Your API key might be invalid or expired
  • You might have run out of credits
  • The provider's server might be down — try another provider

13. ❓ Frequently Asked Questions

Is this tool free?

The software itself is 100% free and open source. But you need API keys for AI services. DeepSeek gives free credits, Pexels is free, and Azure TTS V1 is free. So you can use it for free!

What kind of videos can I make?

Short videos (30-90 seconds) for TikTok, Instagram Reels, YouTube Shorts, educational content, news summaries, product reviews, tips & tricks, and more.

Do I need to know programming?

No! The app has a user-friendly web interface. Just fill in the fields and click buttons.

Can I use my own video clips?

Yes. Select "Local file" as Video Source and upload your MP4, MOV, or image files.

Can I use my own music?

Yes. Place your MP3 files in the resource/songs folder, then type the filename in "Custom Background Music File".

How many videos can I make per day?

Unlimited! There's no limit in the software. But free API services have rate limits (e.g. Pexels allows 200 requests/hour).

Can I edit the video after it's created?

This app creates the final video file. You can download the MP4 and edit it further in any video editing software.

Is my data private?

Yes. Everything runs on your own server/Docker. Your API keys stay on your computer. Nothing is sent to third parties except the API calls to LLM and Pexels/Pixabay.

  • Shuffle: Efek transisi acak
  • FadeIn / FadeOut: Muncul/hilang perlahan
  • SlideIn / SlideOut: Geser masuk/keluar

Rasio Video

  • Potret 9:16 — Untuk TikTok, Instagram Reels, YouTube Shorts
  • Lanskap 16:9 — Untuk video YouTube, presentasi

Durasi Klip

Berapa lama setiap klip video diputar. 3-5 detik disarankan untuk video pendek yang cepat.

Jumlah Video

Hasilkan 1 hingga 5 video sekaligus. Setiap video akan memiliki klip dan musik yang sedikit berbeda.

Pengaturan Video Lanjutan

Encoder Video: Pilih antara:

  • libx264 (CPU) — Bekerja di mana saja, lebih lambat tapi aman
  • h264_nvenc (NVIDIA) — Jauh lebih cepat jika punya GPU NVIDIA
  • h264_amf (AMD) — Lebih cepat dengan kartu grafis AMD
  • h264_qsv (Intel) — Lebih cepat dengan grafis terintegrasi Intel
💡 Tip: Jika tidak yakin, biarkan sebagai libx264. Ini bekerja di semua sistem.

8. 🔊 Pengaturan Audio — Suara & Musik

Masih di panel tengah, di bawah Pengaturan Video.

Server TTS (Text-to-Speech)

Ini yang membacakan naskah Anda dengan suara. Pilihan:

  • Azure TTS V1 (Default) — ✅ Gratis, tanpa setup. Kualitas bagus.
  • Azure TTS V2 — Kualitas lebih baik, tapi perlu API key Azure + region.
  • SiliconFlow TTS — Alternatif TTS, perlu API key dari SiliconFlow.
  • Google Gemini TTS — Menggunakan API key Gemini Anda. Kualitas bagus.
  • Xiaomi MiMo TTS — Dari Xiaomi, perlu API key MiMo.
  • Tanpa Suara — Lewati narasi (video akan diam atau hanya pakai BGM).

Suara Sintesis

Pilih suara. Suara yang tersedia tergantung pada server TTS Anda. Untuk Azure TTS V1, Anda bisa memilih dari banyak bahasa dan gender.

Volume & Kecepatan Suara

  • Volume: 1.0 normal. Turunkan untuk narasi latar.
  • Kecepatan: 1.0 kecepatan normal. 1.2 sedikit lebih cepat.

File Audio Kustom

Unggah audio Anda sendiri (MP3, WAV, dll.). Jika mengunggah, suara TTS akan dilewati dan audio Anda digunakan langsung.

Musik Latar

  • Tanpa Musik Latar: Hanya narasi, tanpa BGM.
  • Musik Latar Acak: Aplikasi memilih lagu acak dari track bawaan.
  • Musik Latar Kustom: Ketik nama file (letakkan MP3 Anda di folder resource/songs).
🎵 Tip musik: Jaga volume musik latar di 0.2-0.3 agar narasi tetap jelas dan mudah didengar.

9. 📝 Pengaturan Subtitle — Menambahkan Teks

Panel kanan mengontrol subtitle (teks yang muncul di layar).

Aktifkan Subtitle

Nyalakan atau matikan subtitle. Jika mati, pengaturan lainnya di bawah tidak berpengaruh.

Font Subtitle

Pilih dari font yang tersedia. Aplikasi sudah dilengkapi beberapa font bawaan.

Posisi Subtitle

  • Bawah: Paling umum — subtitle di bagian bawah layar.
  • Tengah: Di tengah (bagus untuk menonjolkan kata kunci).
  • Atas: Di bagian atas.
  • Kustom: Pilih persentase dari atas (misal 70% = dekat bawah).

Ukuran Font & Warna

  • Ukuran Font: 30 (kecil) hingga 100 (besar). 60 adalah default yang baik.
  • Warna Font: Pilih warna apapun. Putih (#FFFFFF) paling aman.
  • Warna & Ketebalan Garis Tepi: Menambahkan border di sekitar teks agar mudah terbaca.

Latar Belakang Subtitle

Aktifkan latar belakang di belakang teks agar terbaca meskipun di video yang terang.

  • Warna Latar: Biasanya hitam (#000000) dengan transparansi.
  • Latar Membulat: Membuat latar belakang berbentuk pil, bukan persegi panjang.

10. 🚀 Membuat dan Mendownload Video

Sebelum Mengklik Generate

Ceklis:

  • ✅ API key LLM sudah diisi (Basic Settings)
  • ✅ API key Sumber Video sudah diisi (Pexels atau Pixabay)
  • ✅ Topik atau naskah video sudah dimasukkan
  • ✅ Koneksi internet berfungsi

Klik "Generate Video"

Tombol besar di bagian bawah. Proses memakan waktu 1-5 menit tergantung panjang video dan kecepatan internet.

Apa yang terjadi selama pembuatan:

  1. Aplikasi memanggil LLM untuk menulis naskah (jika belum ada)
  2. Aplikasi mencari klip video di Pexels/Pixabay
  3. Aplikasi membuat audio narasi menggunakan TTS
  4. Aplikasi menggabungkan semuanya — klip, audio, subtitle, musik
  5. Aplikasi merender file MP4 video akhir

Setelah pembuatan selesai

Anda akan melihat pemutar video. Anda bisa menonton video langsung di browser. Video juga disimpan di folder storage/tasks/ di server Anda.

11. 💡 Tips Pro & Praktik Terbaik

Menulis Topik yang Baik

  • Spesifik: "Cara membuat kue coklat" lebih baik dari "Kue"
  • Sertakan angka: "5 Tips Teratas" lebih menarik dari "Tips"
  • Tambahkan target: "Untuk pemula" atau "Di tahun 2026" membantu AI fokus

Mendapatkan Suara Narasi Lebih Baik

  • Jaga naskah antara 50-100 kata untuk video 30 detik
  • Gunakan kalimat sederhana — AI membacanya lebih natural
  • Tambahkan tanda baca (titik, koma) — membantu TTS terdengar alami
  • Coba suara yang berbeda — beberapa terdengar lebih natural dari yang lain

Tips Kualitas Video

  • Gunakan Pexels sebagai sumber video — kualitas klip terbaik
  • Atur durasi klip ke 4 detik — tidak terlalu pendek, tidak terlalu panjang
  • Gunakan transisi acak untuk tampilan profesional
  • Jaga volume musik latar di 20-30%
  • Aktifkan subtitle — banyak penonton menonton tanpa suara

Strategi Batch Generation

Atur "Jumlah Video" ke 3-5 untuk membuat beberapa versi. Setiap versi akan memiliki klip dan musik berbeda. Posting di berbagai platform untuk melihat mana yang paling baik.

Menghemat Waktu

  • Simpan API key Anda — aplikasi akan mengingatnya
  • Gunakan ulang naskah — edit sedikit untuk video baru
  • Gunakan pengaturan yang sama — setelah menemukan yang cocok, pertahankan

12. 🔧 Pemecahan Masalah — Masalah Umum

"Please Enter the LLM API Key"

Buka Basic Settings, pastikan Anda sudah memilih LLM Provider dan memasukkan API key yang valid.

"Please Enter the Pexels API Key"

Buka Basic Settings → Video Source Settings, masukkan API key Pexels Anda. Dapatkan gratis di pexels.com/api.

Pembuatan video gagal / macet

  • Periksa koneksi internet Anda
  • Pastikan API key Anda benar
  • Coba topik video yang lebih pendek dengan lebih sedikit kata kunci
  • Kurangi "Jumlah Video" menjadi 1
  • Periksa panel log — ini menunjukkan apa yang terjadi di belakang layar

Subtitle tidak muncul

Pastikan "Enable Subtitles" dicentang di panel kanan.

Tidak ada suara / audio

  • Periksa server TTS sudah dipilih (bukan "Tanpa Suara")
  • Pastikan Anda memilih suara di "Speech Synthesis"
  • Periksa volume suara tidak diatur ke 0

Klip video tidak ditemukan

  • Kata kunci Anda mungkin terlalu spesifik. Coba istilah yang lebih umum.
  • Periksa API key Pexels/Pixabay Anda valid.
  • Ganti ke sumber video yang berbeda.

Pesan "Error from LLM"

  • API key Anda mungkin tidak valid atau kedaluwarsa
  • Kredit Anda mungkin habis
  • Server provider mungkin down — coba provider lain

13. ❓ Pertanyaan yang Sering Diajukan

Apakah alat ini gratis?

Perangkat lunaknya 100% gratis dan open source. Tapi Anda perlu API key untuk layanan AI. DeepSeek memberikan kredit gratis, Pexels gratis, dan Azure TTS V1 gratis. Jadi Anda bisa menggunakannya secara gratis!

Video apa saja yang bisa saya buat?

Video pendek (30-90 detik) untuk TikTok, Instagram Reels, YouTube Shorts, konten edukasi, ringkasan berita, review produk, tips & trik, dan lainnya.

Apakah saya perlu bisa coding?

Tidak! Aplikasi ini memiliki antarmuka web yang mudah digunakan. Cukup isi field dan klik tombol.

Bisakah saya menggunakan klip video sendiri?

Ya. Pilih "Local file" sebagai Sumber Video dan unggah file MP4, MOV, atau gambar Anda.

Bisakah saya menggunakan musik sendiri?

Ya. Letakkan file MP3 Anda di folder resource/songs, lalu ketik nama file di "Custom Background Music File".

Berapa banyak video yang bisa saya buat per hari?

Tidak terbatas! Tidak ada batasan di perangkat lunak. Tapi layanan API gratis memiliki batas (misal Pexels mengizinkan 200 permintaan/jam).

Bisakah saya mengedit video setelah dibuat?

Aplikasi ini membuat file video akhir. Anda bisa download MP4 dan mengeditnya lebih lanjut di software editing video apapun.

Apakah data saya pribadi?

Ya. Semuanya berjalan di server/Docker Anda sendiri. API key Anda tetap di komputer Anda. Tidak ada yang dikirim ke pihak ketiga kecuali panggilan API ke LLM dan Pexels/Pixabay.
